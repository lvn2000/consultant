# Consultant Admin App

Admin panel for the Consultant platform built with Nuxt 3 and Vue 3.

## Features

- 👤 User management
- 📊 Analytics and metrics
- ⚙️ System settings
- 🔐 Role-based access control

## Setup

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Environment Variables

Create `.env` file:

```env
NUXT_PUBLIC_API_BASE=http://localhost:8090/api
```

## Tech Stack

- **Nuxt 3** - Vue framework
- **Vue 3** - UI framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Pinia** - State management
