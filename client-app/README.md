# Consultant Client App

Vue 3 + Nuxt 3 application for the client portal.

## Setup

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Production build
npm run build

# Preview production build
npm run preview

# Type check
npm run type-check

# Lint and fix code style issues
npm run lint:fix

# Run tests
npm run test
```

## Access

- Development: http://localhost:3002
- Backend API: http://localhost:8090/api

## Stack

- **Framework**: Nuxt 3
- **UI Framework**: Vue 3
- **Styling**: Tailwind CSS
- **State Management**: Pinia
- **Icons**: Nuxt Icon
- **Testing**: Vitest
- **Type Safety**: TypeScript

## Features

- Find and book consultations with specialists
- User authentication and profile management
- Rating and review system
- Real-time availability checking
